# Summarise a prLogistic object

Summarise a prLogistic object

## Usage

``` r
# S3 method for class 'prLogistic'
summary(object, ...)
```

## Arguments

- object:

  A `prLogistic` object.

- ...:

  Currently ignored.

## Value

Invisibly returns the `prLogistic` object. Called for its side effect of
printing the model call followed by the formatted estimates.
