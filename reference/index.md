# Package index

## Main functions

Estimate prevalence ratios and confidence intervals

- [`prLogisticDelta()`](https://raydonal.github.io/prLogistic/reference/prLogisticDelta.md)
  : Estimate Prevalence Ratios via Logistic Regression – Delta Method
- [`prLogisticBootCond()`](https://raydonal.github.io/prLogistic/reference/prLogisticBootCond.md)
  : Bootstrap CI for Prevalence Ratios – Conditional Standardisation
- [`prLogisticBootMarg()`](https://raydonal.github.io/prLogistic/reference/prLogisticBootMarg.md)
  : Bootstrap CI for Prevalence Ratios – Marginal Standardisation

## Specialised wrappers

Wrappers for longitudinal and survey data

- [`prLogisticGEE()`](https://raydonal.github.io/prLogistic/reference/prLogisticGEE.md)
  : Prevalence Ratios for Longitudinal Data – GEE Models
- [`prLogisticSurvey()`](https://raydonal.github.io/prLogistic/reference/prLogisticSurvey.md)
  : Prevalence Ratios for Complex Survey Data

## S3 methods

Methods for `prLogistic` objects

- [`print(`*`<prLogistic>`*`)`](https://raydonal.github.io/prLogistic/reference/print.prLogistic.md)
  : Print a prLogistic object
- [`summary(`*`<prLogistic>`*`)`](https://raydonal.github.io/prLogistic/reference/summary.prLogistic.md)
  : Summarise a prLogistic object
- [`coef(`*`<prLogistic>`*`)`](https://raydonal.github.io/prLogistic/reference/coef.prLogistic.md)
  : Extract prevalence ratio point estimates
- [`confint(`*`<prLogistic>`*`)`](https://raydonal.github.io/prLogistic/reference/confint.prLogistic.md)
  : Extract confidence intervals for prevalence ratios
- [`plot(`*`<prLogistic>`*`)`](https://raydonal.github.io/prLogistic/reference/plot.prLogistic.md)
  : Forest plot of prevalence ratios
